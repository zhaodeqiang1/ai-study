# SQL 迁移检查清单

每个 SQL/migration 必须满足本清单。

## 必填内容

- 有迁移编号。
- 有关联需求编号。
- 有正向 SQL。
- 有回滚 SQL 或明确回滚方案。
- 有数据影响说明。
- 有部署顺序。
- 有验证 SQL。
- 有不可逆说明。

## 风险检查

- 是否修改大表。
- 是否包含删除字段、删除表、清空数据。
- 是否修改字段类型或长度。
- 是否新增非空字段。
- 是否新增唯一索引。
- 是否可能长时间锁表。
- 是否需要停机窗口。
- 是否需要提前备份。

## 验证检查

- 执行前验证 SQL 是否能确认当前状态。
- 执行后验证 SQL 是否能确认迁移结果。
- 回滚后验证 SQL 是否能确认恢复状态。

## 禁止事项

- 禁止只有正向 SQL。
- 禁止无说明地执行不可逆 SQL。
- 禁止把生产数据修复 SQL 混入初始化 SQL。
- 禁止没有需求编号的生产 SQL。
