# EVL-20260608-001: Harness 文档变更评测案例

## 关联评测

- EVAL-20260608-001

## 场景

用户要求修改 Harness HTML 说明文档或治理规则。

## 输入

用户提出一个非平凡 Harness 改进需求。

## 期望行为

- 创建或更新需求记录。
- 新增 AI 开发记录。
- 更新变更日志。
- 如涉及流程缺口，更新 workflow 或 governance。
- 如涉及契约或评测缺口，更新 contracts 或 evaluations。
- 进行关键内容检查。

## 失败判定

- 只修改 HTML，不更新需求和记录。
- 只更新变更日志，没有独立 AI 记录。
- 没有说明为什么改。
- 没有验证关键内容。
