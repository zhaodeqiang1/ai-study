# 评测集说明

评测集用于判断 Harness 规则和 AI 功能是否真的可执行。

如果说契约定义“应该满足什么”，评测集定义“用什么案例来检查”。

## 目录

- `datasets`：评测数据集。
- `cases`：单个评测案例。
- `rubrics`：评分标准。
- `INDEX.md`：评测索引。

## 适用范围

- Harness 自身治理质量。
- AI 代码生成过程质量。
- Spring AI Alibaba 功能质量。
- RAG 问答质量。
- SQL 迁移质量。
- 审查门质量。

## 评测结果

评测结果应写入 AI 开发记录：

- 使用的评测集。
- 通过项。
- 失败项。
- 风险。
- 后续改进。
