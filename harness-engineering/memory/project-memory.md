# 项目记忆

本文件记录长期稳定的项目信息，供 AI 在后续任务中少量读取。

## 技术栈

- 后端：Java、Spring Cloud、Spring Cloud Alibaba、RuoYi Cloud。
- 前端：Vue 管理后台。
- 数据访问：MyBatis。
- 构建：Maven。

## 项目偏好

- 保持 RuoYi 原有结构。
- 新 AI 业务优先放在 Harness 独立模块。
- 公共能力只有在多模块确实复用时才下沉。
- 生产能力优先考虑权限、安全、审计和配置。
