# AI 功能契约

Spring AI Alibaba、RAG、模型调用、Prompt、向量检索、工具调用等功能必须满足本契约。

## 配置

- 模型供应商可配置。
- 模型名称可配置。
- API Key 不进入代码。
- 超时、重试、限流可配置或有明确默认值。

## 安全

- 用户输入有长度限制。
- 文件上传有大小、类型和数量限制。
- AI 输出不能直接驱动高风险操作。
- 工具调用必须有白名单和权限检查。

## 可观察性

- 记录调用用户。
- 记录业务场景。
- 记录模型供应商和模型名。
- 记录 Prompt 版本。
- 记录 Token 使用量。
- 记录耗时、成功或失败、错误摘要。

## RAG

- 文档有解析状态。
- 切片有来源信息。
- 检索结果可追踪。
- 回答尽量展示引用来源。
- 用户反馈可记录。

## 评测

AI 功能必须补充或引用评测集，至少覆盖：

- 正常问题。
- 无答案问题。
- 权限边界问题。
- 敏感信息问题。
- 长输入问题。
- 引用来源正确性。
