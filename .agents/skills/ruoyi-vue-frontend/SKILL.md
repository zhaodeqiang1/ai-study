# RuoYi Vue Frontend

Use this repo skill for RuoYi Vue frontend work.

Before editing, read:

- `AGENTS.md`
- `harness-engineering/skills/ruoyi-vue-frontend.md`
- `harness-engineering/contracts/ai-change-contract.md`

For verification, read:

- `harness-engineering/skills/testing-verification.md`

Frontend API files should stay under `ruoyi-ui/src/api/<module>`.

Frontend pages should stay under `ruoyi-ui/src/views/<module>`.

Do not modify `ruoyi-ui/src/permission.js` unless the user explicitly asks or the task cannot be completed without it.
